title: Express middleware

The Express middleware modules listed here are maintained by the
[Expressjs team](https://github.com/orgs/expressjs/people).

| Middleware module                                                                     | Description                                                                                                                                                                             | Replaces built-in function (Express 3) |
| ------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| [body-parser](https://github.com/expressjs/body-parser/blob/master/README.md)         | Parse HTTP request body. See also: [body](https://github.com/raynos/body), [co-body](https://github.com/visionmedia/co-body), and [raw-body](https://github.com/stream-utils/raw-body). | express.bodyParser                     |
| [compression](https://github.com/expressjs/compression/blob/master/README.md)         | Compress HTTP responses.                                                                                                                                                                | express.compress                       |
| [connect-rid](https://github.com/expressjs/connect-rid/blob/master/README.md)         | Generate unique request ID.                                                                                                                                                             | NA                                     |
| [cookie-parser](https://github.com/expressjs/cookie-parser/blob/master/README.md)     | Parse cookie header and populate `req.cookies`. See also [cookies](https://github.com/jed/cookies) and [keygrip](https://github.com/jed/keygrip).                                       | express.cookieParser                   |
| [cookie-session](https://github.com/expressjs/cookie-session/blob/master/README.md)   | Establish cookie-based sessions.                                                                                                                                                        | express.cookieSession                  |
| [cors](https://github.com/expressjs/cors/blob/master/README.md)                       | Enable cross-origin resource sharing (CORS) with various options.                                                                                                                       | NA                                     |
| [errorhandler](https://github.com/expressjs/errorhandler/blob/master/README.md)       | Development error-handling/debugging.                                                                                                                                                   | express.errorHandler                   |
| [method-override](https://github.com/expressjs/method-override/blob/master/README.md) | Override HTTP methods using header.                                                                                                                                                     | express.methodOverride                 |
| [morgan](https://github.com/expressjs/morgan/blob/master/README.md)                   | HTTP request logger.                                                                                                                                                                    | express.logger                         |
| [multer](https://github.com/expressjs/multer/blob/master/README.md)                   | Handle multi-part form data.                                                                                                                                                            | express.bodyParser                     |
| [response-time](https://github.com/expressjs/response-time/blob/master/README.md)     | Record HTTP response time.                                                                                                                                                              | express.responseTime                   |
| [serve-favicon](https://github.com/expressjs/serve-favicon/blob/master/README.md)     | Serve a favicon.                                                                                                                                                                        | express.favicon                        |
| [serve-index](https://github.com/expressjs/serve-index/blob/master/README.md)         | Serve directory listing for a given path.                                                                                                                                               | express.directory                      |
| [serve-static](https://github.com/expressjs/serve-static/blob/master/README.md)       | Serve static files.                                                                                                                                                                     | express.static                         |
| [session](https://github.com/expressjs/session/blob/master/README.md)                 | Establish server-based sessions (development only).                                                                                                                                     | express.session                        |
| [timeout](https://github.com/expressjs/timeout/blob/master/README.md)                 | Set a timeout period for HTTP request processing.                                                                                                                                       | express.timeout                        |
| [vhost](https://github.com/expressjs/vhost/blob/master/README.md)                     | Create virtual domains.                                                                                                                                                                 | express.vhost                          |

## Additional middleware modules

These are some additional popular middleware modules.

{/* {% include community-caveat.html %} */}

| Middleware&nbsp;module                                                          | Description                                                                                                                                         |
| ------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| [cls-rtracer](https://github.com/puzpuzpuz/cls-rtracer)                         | Middleware for CLS-based request id generation. An out-of-the-box solution for adding request ids into your logs.                                   |
| [connect-image-optimus](https://github.com/msemenistyi/connect-image-optimus)   | Optimize image serving. Switches images to `.webp` or `.jxr`, if possible.                                                                          |
| [error-handler-json](https://github.com/mifi/error-handler-json)                | An error handler for JSON APIs (fork of `api-error-handler`.)                                                                                       |
| [express-debug](https://github.com/devoidfury/express-debug)                    | Development tool that adds information about template variables (locals), current session, and so on.                                               |
| [express-partial-response](https://github.com/nemtsov/express-partial-response) | Filters out parts of JSON responses based on the `fields` query-string; by using Google API's Partial Response.                                     |
| [express-simple-cdn](https://github.com/jamiesteven/express-simple-cdn)         | Use a CDN for static assets, with multiple host support.                                                                                            |
| [express-slash](https://github.com/ericf/express-slash)                         | Handles routes with and without trailing slashes.                                                                                                   |
| [express-uncapitalize](https://github.com/jamiesteven/express-uncapitalize)     | Redirects HTTP requests containing uppercase to a canonical lowercase form.                                                                         |
| [helmet](https://github.com/helmetjs/helmet)                                    | Helps secure your apps by setting various HTTP headers.                                                                                             |
| [join-io](https://github.com/coderaiser/join-io)                                | Joins files on the fly to reduce the requests count.                                                                                                |
| [passport](https://github.com/jaredhanson/passport)                             | Authentication using "strategies" such as OAuth, OpenID and many others. See [http://passportjs.org/](http://passportjs.org/) for more information. |
| [static-expiry](https://github.com/paulwalker/connect-static-expiry)            | Fingerprint URLs or caching headers for static assets.                                                                                              |
| [view-helpers](https://github.com/madhums/node-view-helpers)                    | Common helper methods for views.                                                                                                                    |
| [sriracha-admin](https://github.com/hdngr/siracha)                              | Dynamically generate an admin site for Mongoose.                                                                                                    |

For more middleware modules, see [http-framework](https://github.com/Raynos/http-framework#modules).
